## Как читать таблицы

- **Частота** — относительная оценка спроса: **В** (высокая), **С** (средняя),
  **Н** (низкая/нишевая). Это экспертная оценка, не точные цифры —
  **обязательно сверьте по [Яндекс Wordstat](https://wordstat.yandex.ru)**
  (регион Тверь) перед финальным планом. Точных данных по объёму без
  подключённого SEO-инструмента (Ahrefs/Топвизор) нет.
- **Конкуренция** — насколько забита выдача (с учётом маркетплейсов).
- **Приоритет** — где быстрее всего реальный результат для молодого локального
  сайта: **P1** (делать первым), **P2**, **P3**.
- **Страница** — куда вести запрос (существующая или новая).

## Стратегическая рамка

1. **Не воюйте за `купить кухню в твери`.** Топ структурно занят
   маркетплейсами (Ozon, Avito, Я.Маркет) и федералами (Hoff, Lemana, Мария).
   Интент там — «готовая кухня со склада», у вас — «на заказ». Это чужая битва.
2. **Ваше поле — `кухни на заказ`, `от производителя`, `по размерам`,
   по стилям и по городам области.** Там конкуренты — локальные тверские
   фабрики, которых реально догнать.
3. **Низкочастотные запросы (НЧ) — ваш быстрый трафик.** Молодой сайт сначала
   ранжируется по длинным точным запросам, потом — по средним.

## Краткий план на первые недели

1. **P1-страницы каталога:** добавить `угловые кухни` (и при возможности
   `классику` + `белую кухню`) — низкая конкуренция, явный коммерческий интент.
2. **P1-статьи:** «готовая или на заказ», «маленькая кухня», «сроки» — закрывают
   частые сомнения покупателя и перехватывают часть «купить»-трафика.
3. **Уникализировать тексты городских лендингов** (если есть дубли) и добавить
   локальные отзывы.
4. После публикации — отправлять новые URL на переобход в Вебмастере (см.
   `yandex-webmaster-search-console.md`).

> Перед запуском плана сверьте частоты по Wordstat (регион Тверь) и при
> возможности подключите Топвизор/Ahrefs — тогда таблицы можно дополнить
> точными цифрами и позициями.
