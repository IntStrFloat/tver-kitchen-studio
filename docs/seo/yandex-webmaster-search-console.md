# Подключение Яндекс.Вебмастера и Google Search Console — пошагово

Цель: ускорить индексацию kuhnitver.ru и начать получать данные о позициях.
Коды верификации уже вшиты в сайт (`src/app/layout.tsx` → `verification`), поэтому
подтверждение прав займёт пару кликов.

Порядок важности: сначала **Яндекс.Вебмастер** (основной трафик в РФ), затем
**Search Console**, затем гео-карточки (Яндекс.Бизнес / 2ГИС / Google Бизнес).

---

## Часть 1. Яндекс.Вебмастер (15 минут)

Адрес: https://webmaster.yandex.ru

1. **Войдите** под тем Яндекс-аккаунтом, который будет «рабочим» для бизнеса
   (на него же потом заведёте Яндекс.Бизнес и Метрику — лучше один аккаунт).

2. Нажмите **«+ Добавить сайт»** (правый верхний угол).
   - Введите `https://kuhnitver.ru` → **«Добавить»**.

3. **Подтверждение прав.** Откроется страница с тремя вкладками
   (Мета-тег / HTML-файл / DNS). Выберите вкладку **«Мета-тег»**.
   - Яндекс покажет строку вида `<meta name="yandex-verification" content="…">`.
   - Этот тег **уже стоит на сайте** (значение `ce67eb671a8c4f20`). Ничего
     вставлять не нужно — просто нажмите **«Проверить»**.
   - Если проверка не прошла: убедитесь, что на проде задеплоена свежая версия
     (тег отдаётся в `<head>` главной — проверено, он там есть). Подождите
     минуту и нажмите «Проверить» ещё раз.

4. **Добавьте Sitemap.** Левое меню → **«Индексирование» → «Файлы Sitemap»**.
   - В поле вставьте `https://kuhnitver.ru/sitemap.xml` → **«Добавить»**.
   - Статус сменится на «Обрабатывается», через несколько часов/дней — на
     «Проверен» с числом найденных URL (должно быть ~38).

5. **Отправьте страницы на переобход** (это ускоряет первую индексацию).
   Левое меню → **«Индексирование» → «Переобход страниц»**.
   - Вставляйте по одному URL и жмите **«Отправить»**. Лимит обычно 10–20 в день.
   - Приоритетные URL в первую очередь:
     ```
     https://kuhnitver.ru/
     https://kuhnitver.ru/catalog
     https://kuhnitver.ru/price
     https://kuhnitver.ru/kuhni/tveri
     https://kuhnitver.ru/portfolio
     https://kuhnitver.ru/about
     https://kuhnitver.ru/blog
     ```
   - Остальные города/стили/статьи отправьте на следующий день.

6. **Укажите регион — это критично для запросов «в Твери».**
   Левое меню → **«Информация о сайте» → «Региональность»**.
   - Добавьте регион **Тверь**. Если попросит подтверждение — привяжите
     карточку Яндекс.Бизнеса (см. Часть 3) или укажите страницу с адресом
     (`/contacts`).

7. **Проверьте «Диагностику».** Левое меню → **«Диагностика» → «Диагностика
   сайта»**. Устраните «фатальные» и «критичные» замечания, если появятся
   (по текущему коду их быть не должно).

8. **Включите обходчик с учётом Метрики** (опционально, помогает): счётчик
   Яндекс.Метрики на сайте уже стоит (id `106971287`) — Вебмастер сам
   подтянет данные, отдельных действий не требуется.

**Что мониторить дальше:** «Индексирование → Страницы в поиске» (сколько URL
в индексе) и «Поисковые запросы → Последние запросы» (по чему уже показываетесь).

## Часть 2. Google Search Console (10 минут)

Адрес: https://search.google.com/search-console

1. **Войдите** под Google-аккаунтом бизнеса.

2. Слева вверху выпадающий список ресурсов → **«+ Добавить ресурс»**.
   Появятся два варианта:
   - **«Ресурс с префиксом URL»** ← выберите этот (проще верификация).
   - Введите `https://kuhnitver.ru/` (со слешом в конце) → **«Продолжить»**.

3. **Подтверждение.** В списке способов выберите **«HTML-тег»**.
   - Google покажет `<meta name="google-site-verification" content="…">`.
   - Тег **уже на сайте** (значение `99aXy2XdjB2Jw09DwzV43zwq7ZxIyvvJJw4Fz1zR2NA`).
     Просто нажмите **«Подтвердить»**.

4. **Добавьте Sitemap.** Левое меню → **«Файлы Sitemap»** (Sitemaps).
   - В поле «Добавление файла Sitemap» введите `sitemap.xml` → **«Отправить»**.
   - Статус должен стать «Успешно», обнаружено ~38 URL.

5. **Запросите индексацию вручную для топ-страниц.** Сверху строка **«Проверка
   любого URL»** (URL Inspection).
   - Вставьте `https://kuhnitver.ru/` → Enter → дождитесь отчёта →
     **«Запросить индексирование»**.
   - Повторите для `/catalog`, `/price`, `/kuhni/tveri`, `/blog`,
     `/portfolio` (по несколько штук в день).

6. **Что мониторить:** левое меню → **«Индексирование → Страницы»** (сколько
   проиндексировано / почему исключено) и **«Эффективность»** (клики, показы,
   средняя позиция, запросы).

> Примечание про www: канонический адрес — без www (`https://kuhnitver.ru/`).
> Редирект http→https настроен. Отдельный ресурс на `www` в Search Console
> заводить не нужно — canonical уже указывает на основной адрес.

## Часть 3. Гео-карточки (самый высокий ROI для «в Твери»)

Для локальных запросов Яндекс и Google сильно опираются на карточки
организаций (адрес, отзывы, фото). У вас есть реальный адрес — заведите
карточки везде. Это часто даёт результат быстрее, чем органика.

1. **Яндекс.Бизнес** — https://business.yandex.ru
   - Добавьте организацию: название «Kuhnitver», категория «Изготовление
     мебели / Кухни на заказ», адрес **г. Тверь, ул. Коминтерна, 95,
     ТЦ «Мебельный»**, телефон, часы (Пн–Сб 10:00–19:00), сайт.
   - Подтвердите права (звонок/СМС/письмо).
   - Загрузите фото производства, шоурума, работ. Привяжите карточку к
     Вебмастеру (Часть 1, шаг 6).

2. **2ГИС** — https://account.2gis.com (добавление организации бесплатно).
   Те же данные. В Твери 2ГИС популярен — это дополнительный канал заявок.

3. **Google Business Profile** — https://business.google.com
   Те же данные. Появитесь на Google Картах и в локальной выдаче.

4. **Отзывы.** Как только карточки активны — попросите 5–10 первых клиентов
   оставить отзыв на Яндексе и 2ГИС. Это запускает и рейтинг, и поведенческие
   сигналы. (Важно: не выдумывать отзывы — за фейки и Яндекс, и Google
   наказывают; в коде `aggregateRating` намеренно не проставлен до появления
   реальных отзывов — это правильно.)

---

## Чего ожидать по срокам (честно)

- **3–7 дней:** Яндекс и Google начнут заходить чаще, число страниц в индексе
  пойдёт вверх.
- **2–4 недели:** большинство из 38 URL в индексе; появятся первые показы по
  низкочастотным запросам («кухни лофт тверь», города области).
- **2–4 месяца:** молодой домен начинает набирать видимость по средним
  запросам — при условии, что параллельно растут ссылки, отзывы и поведенческие.

Переобход и Sitemap ускоряют **индексацию**, но не отменяют «отлёжку» молодого
домена в **ранжировании**. Это нормально.

## Часть 4. Фид услуг для блока «Услуги» в Поиске (бесплатный трафик)

В Поиске Яндекса есть отдельный блок услуг. Туда загружается YML-фид —
он уже сгенерирован сайтом и доступен по адресу:

```
https://kuhnitver.ru/uslugi.yml
```

Как подключить: **Вебмастер → «Фиды»** → категория бизнеса **«Исполнители»**,
регион **Тверь, Тверская область** → в поле «Ссылка на файл с предложениями»
вставьте адрес выше → отправьте. Проверка фида занимает **до недели**.

Что внутри (9 предложений, генерируются из каталога автоматически):
- 7 типов кухонь на заказ (по стилям) со ссылками на страницы каталога и
  ценами «от …»;
- замер кухни (1 500 ₽, вычитается из заказа);
- доставка и установка (бесплатно по Твери).

Поля рейтинга и числа отзывов сейчас = `0` (честно, до накопления реальных
отзывов). После подключения Яндекс.Бизнеса и появления отзывов их можно
подставить в фид — править `src/app/uslugi.yml/route.ts` (константы
`RATING`, `REVIEWS_COUNT`, `FOUNDED_YEAR`).

> Фид пересобирается при каждом деплое из тех же данных, что и каталог, —
> новые стили и цены попадают в него автоматически.
