# Требования к изображениям каталога примерки

- Формат: WebP или JPEG.
- Минимальный размер: 1600 px по длинной стороне.
- Изображения не должны содержать людей или чувствительный контент.
- Для каждого изделия нужны 2-4 чистых ракурса.
- Имена файлов: `<product-id>-01.webp` ... `<product-id>-04.webp`.
- Изображения должны четко показывать фасад, цвет и фурнитуру. Предпочтителен простой или прозрачный фон.

Пустой каталог допустим только как оболочка для разработки. До пилотного или публичного использования необходимо подготовить 6-12 реальных изделий для каждой группы.
